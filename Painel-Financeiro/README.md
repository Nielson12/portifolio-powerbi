# Dashboard Financeiro

Análise detalhada da saúde financeira corporativa para suporte à decisão estratégica.

## Indicadores em Destaque
- **Receita e Despesas:** Monitoramento do fluxo mensal.
- **Lucro e Margem:** Visão consolidada da rentabilidade.
- **Fluxo de Caixa:** Controle de entradas e saídas.

## Tecnologias
- Power BI, DAX, Excel.
