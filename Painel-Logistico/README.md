# Dashboard Logístico

Monitoramento de indicadores de desempenho (KPIs) para otimização da cadeia de suprimentos e logística de entregas.

## Indicadores em Destaque
- **Eficiência de Entregas:** Percentual de entregas realizadas no prazo.
- **Gestão de Atrasos:** Análise das causas raiz de gargalos operacionais.
- **Custos de Frete:** Monitoramento de despesas por região e transportadora.

## Tecnologias
- Power BI, DAX, Excel.
